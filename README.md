## 📖 Overview

**Portfoly** is a sleek and responsive portfolio builder that enables users to browse premium portfolio templates, preview designs, and customize their portfolio through an elegant editor. Built using pure HTML, CSS, and JavaScript, the project focuses on delivering a premium user experience without relying on external frameworks.

---

## ✨ Features

- 🎨 Modern and premium UI/UX
- 📱 Fully responsive design
- 🌙 Elegant dark theme
- 🖼️ Portfolio template gallery
- ⚡ Interactive portfolio editor
- 🔍 Live preview experience
- 🎭 Smooth animations and transitions
- 📂 Template categorization
- 🚀 Fast and lightweight
- 💻 Built using Vanilla JavaScript
